# Nexus AI Dashboard 🚀
Professional AI Assistant powered by Google Gemini 1.5 Flash.

## Features
- **Ultra-Fast Responses:** Powered by Gemini Flash.
- **Dark Mode Dashboard:** Pro UI for interactions.
- **Secure Architecture:** No hardcoded tokens.
- **Cloud Ready:** Optimized for Render.com.

## Deployment Setup
1. Fork this repo to your GitHub.
2. Go to Render.com -> New Web Service.
3. Connect your Repo.
4. Set Environment Variable: `GEMINI_API_KEY` = (Aapki Google AI Key).
5. Deploy!
